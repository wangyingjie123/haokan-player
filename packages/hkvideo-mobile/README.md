## hkvideo-mpbile
用于移动端h5视频播放
### 开发
```
1、cd packages/hkplayer-mobile
2、npm i
3、npm run dev
```
### 使用
```
import Player from 'hkplayer-mobile-mpbile';
const player = new Player({
    id: 'app',
    cssFullscreen: true,
    fluid: true, // 流式布局
    // autoplay: true, // 自动播放
    url: 'https://s1.pstatp.com/cdn/expire-1-M/byted-player-videos/1.0.0/xgplayer-demo.mp4',
});
```
